## Steps to create the project

### Clone the git repo

```sh
git clone https://github.com/Ladalo-Daniel/wetin-dey-sup.git
```

Install the project dependencies with npm

```sh
npm install
```

Start the development server by running

```sh
npm run dev
```

See your changes live.
